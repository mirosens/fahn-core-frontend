# FAHN-CORE Frontend (Next.js App Router)

- Next.js (App Router, TS strict)
- Env-Validation via @t3-oss/env-nextjs
- URL-state via nuqs
- Vitest + Testing Library
- Husky + lint-staged (pre-commit)

## Setup

```bash
pnpm install
cp .env.example .env.local
# Werte anpassen

pnpm lint
pnpm build
pnpm test
pnpm dev
```
